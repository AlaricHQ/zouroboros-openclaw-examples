# Autoloop Example

Minimal starter for `zouroboros-autoloop`.

## Install

```bash
npm install -g zouroboros-autoloop
```

## Files

- `program.md` — objective, target file, experiment command, and constraints

## Run

```bash
autoloop --program ./program.md --executor "openclaw ask"
```

## Notes

- Run inside a git repo
- Keep the metric mechanical and easy to measure
- Start with one target file and one command
